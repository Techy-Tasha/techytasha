Command Reference

  • pwd - print working directory
  • ls - list everything inside the current directory
  • ls -l - lists directory vertically, and alphabetically, also shows permissions, date and file creator, and more.
  • cd ~ - takes me back to the home directory
  • chmod - adjusts permissions
  • mkdir - make a new directory
  • touch - create a new file
  • rm - delete file
  • rm-r - remove directory and all files
  • mv - move
  • cp - copy
